## 网络
查看端口是否监听 
  
```
netstat -an | grep 3000
```

查看端口对应的PID，参考[这里](https://tonydeng.github.io/2016/07/07/use-lsof-to-replace-netstat/)  

```
lsof -i :3000

lsof -nP -iTCP:3000
```

查看路由  

```
netstat -rn
```

## 磁盘
### 查看所有磁盘和分区：  

```
# 在macOS中fdisk命令没有-l选项
diskutil list
```

### 查看分区挂载信息：  

```
# 和Linux中是一样的
mount
```
    
> 在macOS中，移动硬盘或者U盘默认挂载在/Volumes目录下。

### 卸载移动磁盘
先使用mount找到挂载的路径path，使用diskutil命令卸载。比如：  

```
diskutil unmount path
```

# brew
[brew](http://brew.sh/)是OS X上的包管理软件.
## 安装
```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```
## 配置国内源

```
# vi ~/.zshrc, 添加下面一行
export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.ustc.edu.cn/homebrew-bottles

source ~/.zshrc
```

## 使用
```
# 列出所有已经安装了的包
brew list

# 搜索libvirt包
brew search libvirt

# 查看包的详细信息
brew info libvirt

# 安装包
brew install libvirt

# 卸载包
brew uninstall libvirt

# 升级brew自己的版本
brew update

# 升级某个包的版本，比如ansible
brew upgrade ansible

# 查看所有的service
brew services list

# 启动并开机启动MongoDB
brew services start mongodb

# 停止并在开机时不启动MongoDB
brew services stop mongodb

# OS X上常见的工具包

htop

node

```

# 配置
## Finder
### 修改文件默认打开方式
如果在文件上右击,选择打开方式,只能选择这个文件的打开方式。  
如果想修改一类文件的打开方式,打开一个文件的现实简介窗,在打开方式中选择一个程序,然后选择全部更改。

### Finder默认在当前文件夹
打开Finder偏好设置，切换到高级选项卡，将”执行搜索时：“切换到”搜索当前文件夹“

### 控制Finder显示隐藏文件
打开terminal 

```
# 显示所有文件
defaults write com.apple.finder AppleShowAllFiles YES
killall Finder

# 不显示隐藏文件
defaults write com.apple.finder AppleShowAllFiles NO
killall Finder
```
